# 这里主要讲的是 Apollo Server

Apollo GraphQL 平台，通过 GraphQL 帮助您从 0 到生产能力

# Apollo 平台

Apollo 是 GraphQL 的实现， 旨在满足构建先的， 数据驱动的应用程序的产品团队的需求。

  • 敏捷增量开发
  • 特别注意避免对现有 API 和服务器进行任何更改
  • 强调工具和工作流程

# 最好的处理

将 Apollo 作为服务器和应用程序之间的堆栈中的新层。

# 主要的组成

  • distruted GraphQL(分布式 GraphQL)
    • GraphQL gateway and query execution engine
  • workflow and policy (工作流和政策)
    • Data graph access control
    • Schema change control
  • GraphQL services (GraphQL 服务)
    • Client registry
    • Operation registry
    • Trace warehourse
  • Core open source libraries and tools (核心开源，库和工具)
    • Apollo Client (JavaScript)
    • Apollo Client (native mobile)
    • Apollo Server
    • Apollo Cli
    • Apollo IDE plugins
    • Apollo Chrome devtools
  • central schema registry (重要寄存器)
    • Schema registry

# 副组成

  • Commercial plugins
  • Cloud services
  • Open source libraries

# 核心开源组件

  • Apollo 服务器是 JavaScript GraphQL 服务器
  • Apollo 客户端
  • Apollo 社区贡献的 iOS 和 Android 客户端查询 GraphQL API 成为可能。
  • Apollo Cli 是一个简单的命令行客户端，提供对 Appllo 云服务访问

# 云服务

  • 架构注册表

  • 操作注册表
  • 跟踪仓库
