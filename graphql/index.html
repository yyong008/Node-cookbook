# JavaScript 相关的 GraphQL.js

GraphQL 是 API 的查询语言，也是使用为数据定义的类型系统执行查询的服务器端运行时。GraphQL 不绑定到任何特定的数据库或存储引擎，而是由现有代码和数据支持。

其实 GraphQL 实现不止有 JavaScript, 我们通过官方网站实现得知还有其他的语言也实现了：

  • C# / .NET
  • Clojure
  • Elixir
  • Erlang
  • Go
  • Groovy
  • Java
  • JavaScript
  • Kotlin
  • PHP
  • Python
  • Ruby
  • Rust
  • Scala
  • Swift

# 不同的形式

GraphQL 是一种查询语言的 API

  • 类型形式

描述你所需要的数据

type Project {
  name: String
  tagline: String
  constructor: [User]
}
  • 函数形式

询问你想要的

{
  project(name: "GraphQL") {
    tagline
  }
}
  • 获取可预见的数据
{
  "project": {
    "tagline": "A query language for APIs"
  }
}

# JavaScript

# 安装

npm install graphql

# yarn
yarn add graphql

# demo

var { graphql, buildSchema } = require('graphql')

var schema = buildSchema(`
  type Query {
    hello: String
  }
`)

var root = {
  hello: () => 'Hello world!',
}

graphql(schema, '{ hello }', root).then(response => {
  console.log(response)
})

graphql 的内容入门我们需要 3 个东西：

  • schema
  • 数据源 root
  • 查询对象字符串

# demo express

基于 Express webserver 服务器的一个 GraphQL API 服务端参考实现， 你可以用它结合常规 Express webServer 运行 GraphQL, 也可以作为独立 GraphQL 服务器：

# 安装

yarn add express express-graphql graphql

创建一个简单 express 服务

var express = require('express')
var graphqlHTTP = require('express-graphql')
var { buildSchema } = require('graphql')

var schema = buildSchema(`
  type Query {
    hello: String
  }
`)

var root = {
  hello: () => 'Hello world',
}

var app = express()

app.use(
  '/graphql',
  graphqlHTTP({
    schema: schema,
    rootValue: root,
    graphiql: true,
  })
)

app.listen(4000, () => console.log('Now browse to localhost:4000/graphql'))

# apollo

nodejs 框架 Apollo 的一套 GraphQL server 包，用于 Node.js HTTP 框架如：

  • Express
  • Connect
  • Hapi
  • Koa

# 安装

yarn add apollo-server-express

然后使用 node server.js 运行 server.js

const express = require('express')
const { ApolloServer, gql } = require('apollo-server-express')
const typeDefs = gql`
  type Query {
    hello: String
  }
`

const server = new Apollo({
  typeDefs,
  resolves,
})

const app = express()
server.applyMiddleware({ app })
app.listen({ port: 4000 }, () => {
  console.log('Now browse to http://localhost:4000' + server.graphqlPath)
})
